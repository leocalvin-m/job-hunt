Full Stack Software Developer

Overview
Responsibilities
  • Develop and maintain software applications using NodeJS, JavaScript or TypeScript, ReactJS, and NextJS.
  • Design, create and maintain APIs that integrate with various systems and platforms.
  • Design and develop database schema and queries (SQL and NoSQL).
  • Collaborate with the product team to define and implement new features.
  • Write clean, efficient, maintainable and well-documented code that meets the highest industry standards.
  • Participate in code reviews to ensure code quality and adherence to best practices.
  • Participate in testing activities.
  • Implement automated tests.
  • Participate in the process of the solution deployment to the environment.
  • Troubleshoot and debug software issues to ensure optimal performance.
  • Participate in the production support activities.
  • Continuously improve and optimize security and performance of web applications.
  • Stay up-to-date with emerging trends and technologies in software development.
Required Skills and Experience
  • Proven experience as a Full Stack Software Developer, with a strong understanding of NodeJS, JavaScript or TypeScript, ReactJS, NextJS.
  • Strong understanding of REST API design, implementation and management.
  • Strong understanding of database design and management, including SQL and NoSQL databases.
  • Strong understanding of web development best practices (responsive design, SEO, security, performance, web accessibility).
  • Strong understanding of code quality best practices.
  • Experience with modern software development methodologies, including Agile, Scrum, and DevOps.
  • Strong problem-solving and analytical skills.
  • Strong verbal and written communication skills.
  • Ability to work independently and as part of a team in a fast-paced, dynamic environment.
Nice to have Skills and Experience
  • Ability to use Atlassian tools (Jira, Confluence).
  • Experience with Headless CMS solutions.
  • Experience with Amazon Web Services cloud (serverless architectures like AWS Lambda).
  • Experience with CI/CD pipelines configuration.
